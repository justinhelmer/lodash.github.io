------

The lodash website.

### setup

> only works for `OSX` or `Ubuntu` (`14.04` officially supported).

```bash
npm install
```

### launch

```bash
npm start
```

### update $PATH (optional)

```bash
npm link
```

### build/serve/etc.

```bash
lodocs # will display man page
```
